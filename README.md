# Dark Moon Whisky Website (Guedes Solutions)

Bem-vindo ao repositório do site **Dark Moon Whisky**. Este é um projeto de frontend focado em criar uma experiência visual premium, luxuosa e envolvente para uma marca de whisky.

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias modernas de desenvolvimento web:

- **[Next.js 14 (App Router)](https://nextjs.org/)**: Framework React utilizado para a base estrutural do projeto.
- **[React](https://react.dev/)**: Biblioteca para a criação dos componentes de interface do usuário.
- **[Node.js](https://nodejs.org/)**: Ambiente de execução JavaScript, necessário para rodar o Next.js e gerenciar pacotes via npm.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para a construção do design system e estilização responsiva.
- **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca para criar animações fluidas baseadas no scroll e nas interações do usuário.
- **[TypeScript](https://www.typescriptlang.org/)**: Superset do JavaScript que adiciona tipagem estática ao código.

## Como Executar Localmente

Primeiro, certifique-se de instalar as dependências:

```bash
npm install
```

Em seguida, inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) com o seu navegador para ver o resultado.
